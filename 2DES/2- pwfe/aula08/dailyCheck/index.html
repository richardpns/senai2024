Como Funciona o Gerenciador de Tarefas?

O gerenciador de tarefas do Daily Check permite que você organize suas tarefas diárias de forma eficiente. Basta adicionar as suas tarefas, juntamente com detalhes como nome, data de início, data de término e uma breve descrição. Você pode visualizar suas tarefas na página principal e gerenciar facilmente seu progresso.

Exemplos de Tarefas

Tarefa 1

Lavar o carro cedo

Data de Início: 01/04/2024

Data de Término: 02/04/2024

Tarefa 2

Passear com o cachorro as 18:00

Data de Início: 03/04/2024

Data de Término: 03/04/2024

Tarefa 3

Treinar as 20:00

Data de Início: 03/04/2024

Data de Término: 03/04/2024
